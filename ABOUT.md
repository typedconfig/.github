# .github Directory for TypedConfig (Tyco)

This directory contains community health files and shared resources for the TypedConfig (Tyco) project. These files are automatically used by GitHub to provide information and standards across all repositories in the organization.

## Files

- `README.md`: Home page for the TypedConfig GitHub organization, describing the project, its goals, and key features.
- (Optional) `CONTRIBUTING.md`: Contribution guidelines for all Tyco projects.
- (Optional) `CODE_OF_CONDUCT.md`: Code of conduct for the community.
- (Optional) `SUPPORT.md`: How to get help or support.
- (Optional) `SECURITY.md`: Security policy and vulnerability reporting.

If you would like to contribute, please see the [README](./README.md) and the contributing guidelines (if present).
