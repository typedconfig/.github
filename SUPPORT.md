# Support for TypedConfig (Tyco)

If you need help with Tyco or any of the language bindings, please use the following resources:

- **Documentation:** https://typedconfig.com/docs
- **GitHub Issues:** Use the Issues tab in the relevant repository for bug reports or feature requests.
- **Discussions:** For general questions or design discussions, use GitHub Discussions or open an issue.
- **Email:** info@typedconfig.com

For security-related issues, please see [SECURITY.md](./SECURITY.md) if present, or email us directly.
