# Contributing to TypedConfig (Tyco)

Thank you for your interest in contributing to TypedConfig (Tyco)!

## How to Contribute

- **Bug Reports & Feature Requests:** Please use the Issues tab in the relevant language binding or the main organization repo.
- **Pull Requests:** Fork the relevant repository, create a feature branch, and submit a pull request with a clear description.
- **Discussions:** For general questions, ideas, or design discussions, use GitHub Discussions or open an issue.

## Code Style
- Follow the conventions of the language binding you are contributing to.
- Write clear, concise commit messages.
- Add tests for new features and bug fixes.

## Community
- Be respectful and inclusive.
- See our [Code of Conduct](./CODE_OF_CONDUCT.md) for more details.

## Getting Help
- Website: https://typedconfig.com
- GitHub: https://github.com/typedconfig
- Docs: https://typedconfig.com/docs
